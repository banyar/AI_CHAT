Google Gemini Flash Lite — Free ဆိုလား? Project မှာ ဘယ်လို သုံးမလဲ
✅ Free Tier ရှိတယ် — ဒါပေမဲ့ Limit ရှိတယ်
Model	Free Tier	Paid Price
gemini-2.0-flash-lite	✅ Free	$0.075 / 1M tokens
gemini-2.5-flash-lite	✅ Free	$0.10 / 1M tokens
Free Tier Limits:


Requests per Day    : 1,000 RPD
Requests per Minute : 15 RPM
Tokens per Minute   : 250,000 TPM
⚠️  Free tier မှာ Google က data သုံး improve လုပ်ပိုင်ခွင့် ရတယ်
Customer data (phone, package info) ပါတဲ့ production app မှာ Free tier မသင့်တော်ဘူး — Privacy issue ဖြစ်နိုင်တယ်

Project မှာ သုံးဖို့ ဘာလိုမလဲ
Step 1 — API Key ရယူ

1. https://aistudio.google.com သွားပါ
2. "Get API Key" နှိပ်ပါ
3. Key ကို copy ပါ → .env မှာ ထည့်ပါ
.env မှာ ထည့်:


GEMINI_API_KEY=AIzaSyXXXXXXXXXXXXXXXXXXXXXX
GEMINI_MODEL=gemini-2.0-flash-lite
Step 2 — Project Integration ရှင်းလင်းချက်
သင့် project မှာ LLM layer က internal/llm/ollama.go မှာ ရှိတယ်။ Gemini သုံးဖို့ internal/llm/gemini.go အသစ် တည်ဆောက်ရမယ်။


ယခု Architecture:
    agent.go → llm/ollama.go → Ollama (local)

Gemini ထည့်ပြီးနောက်:
    agent.go → llm/ollama.go  → Ollama (local)
             → llm/gemini.go  → Google Gemini API (cloud)
Step 3 — Gemini API Call ပုံစံ
Ollama နဲ့ ကွာချက်:

Ollama (ယခု)	Gemini API
URL	http://localhost:11434	https://generativelanguage.googleapis.com
Auth	မလို	Authorization: Bearer API_KEY
Format	Ollama JSON	Gemini JSON (ကွာတယ်)
Internet	မလို (local)	လိုတယ်
Cost	Free (local PC)	Free/Paid tier
Ollama vs Gemini — Frontiir Project အတွက် ဘယ်ဟာ ကောင်းမလဲ

Ollama (ယခု သုံးနေတာ)               Gemini API
├── Local PC မှာ run              ├── Cloud (internet လို)
├── Free (unlimited)              ├── Free (1000 req/day limit)
├── Myanmar data private          ├── ⚠️  Data Google ဆီ ရောက်တယ်
├── Qwen 2.5 → မြန်မာ ကောင်း     ├── မြန်မာ support ကောင်းတယ်
└── Internet မလို                 └── Internet ကျသွားရင် ရပ်တယ်
ကျွန်တော့် Recommendation
Use Case	သင့်တော်တဲ့ Model
Development / Testing	Gemini Free ✅ (ကောင်းတယ်)
Production (Customer data ပါ)	Ollama + Qwen 2.5 ✅ (safe)
Demo / Presentation	Gemini Free ✅
Low RAM PC (8GB အောက်)	Gemini Free ✅ (local model မ run နိုင်)
Gemini ထည့်ချင်ရင်
ကျွန်တော် internal/llm/gemini.go ဖန်တီးပေးနိုင်တယ် — Ollama နဲ့ same interface ဖြစ်အောင် ရေးပြီး .env မှာ switch လုပ်နိုင်အောင် လုပ်ပေးမည်:


# .env မှာ ပြောင်းရုံပဲ
LLM_PROVIDER=gemini   # or: ollama
